# 🎉 The Surprise Events – Website

This project is a live website built for a real event decoration business.

🔗 Live Website: https://thesurpriseevents.vercel.app

## 📌 About the Project

This website was created for a friend’s event decoration business to showcase their services like balloon decorations, surprise setups, and event planning.

The goal was simple:

* Create a clean and attractive website
* Make it easy for customers to explore services
* Provide a way to view event photos and ideas

Even though I come from a computer engineering background, this project was built using modern AI tools and platforms instead of writing everything from scratch.

## ⚙️ Tech Stack / Tools Used

* Frontend: HTML, CSS, JavaScript (generated and customized)
* Backend & Storage: Supabase
* Deployment: Vercel
* AI Tools: Claude AI, Antigravity

## 🚀 Features

* 📸 Gallery of event decorations
* 🎈 Different categories of event setups
* ⚡ Fast and responsive UI
* ☁️ Cloud-based image storage using Supabase
* 🌐 Live deployment on Vercel

## 💡 What I Learned

* How to take an idea and turn it into a live product
* Working with AI tools to speed up development
* Connecting frontend with backend services like Supabase
* Deploying real-world projects using Vercel
* Understanding project structure without relying fully on code writing

## 🧠 Note

This project was not built by writing code line-by-line manually. Instead, I used AI tools effectively and focused on:

* Understanding the flow
